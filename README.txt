# AI-Powered Text Analysis with LangChain & Google Gemini

This project leverages the power of **LangChain**, **Google Gemini LLM**, and **prompt engineering** to perform advanced natural language processing tasks including:

- Sentiment Analysis  
- Threat Word Detection  
- Named Entity Recognition (NER)

Built with modularity and clarity in mind, this app is ideal for analyzing user-generated content, social media posts, or any text data requiring intelligent interpretation.

---

## Features

- **Sentiment Analysis**: Classifies text as Positive, Negative, or Neutral using LLM-based reasoning.
- **Threat Word Detection**: Identifies potentially harmful or alarming language using custom prompt templates.
- **Named Entity Recognition**: Extracts entities like names, locations, organizations, and moreâ€”powered by Gemini's contextual understanding.
- **LangChain Integration**: Uses LangChain's `PromptTemplate` and `LLMChain` for structured, reusable logic.
- **Streamlit UI**: Interactive frontend for real-time analysis.
